# **App Name**: BloomBites

## Core Features:

- Product Catalog: Display pre-made snack bouquets with images, descriptions, and prices.
- Custom Bouquet Builder: Allow customers to create custom snack bouquets by selecting from available items and seeing a live price calculation. The description entered by the user for each item is added as a tool, letting the user indicate constraints such as packing style.
- Shopping Cart: Enable customers to add, remove, and update items in their cart.
- Checkout Process: Secure checkout process with customer details, address, payment options (Cash on Delivery), and order summary.
- Admin Dashboard: Provide a secure admin dashboard for managing bouquets, items, and orders.
- User Authentication: Role-based authentication (Admin/Customer) with email/password login.
- Order Management: Allow admins to view and update customer order statuses.

## Style Guidelines:

- Primary color: Gold (#FFD700) to evoke feelings of warmth, celebration, and a premium experience.
- Background color: Very light beige (#F5F5DC) to provide a warm, neutral backdrop.
- Accent color: Dark chocolate brown (#3E2723) for a luxurious, sophisticated feel.
- Headline font: 'Playfair', serif, for an elegant, fashionable, high-end feel; use 'PT Sans' for body.
- Note: currently only Google Fonts are supported.
- Use elegant icons to represent different snack categories and bouquet customization options.
- Clean, modern, and responsive layout with a focus on visual presentation of snack bouquets.
- Subtle animations to enhance user experience, such as smooth transitions and loading indicators.
